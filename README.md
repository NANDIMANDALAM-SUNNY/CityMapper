# CityMapper

CityMappper is a MERN project where users can register and review the places across the globe easily
##deployed url : https://citymapper.netlify.app/

## Installation

git clone https://github.com/NANDIMANDALAM-SUNNY/citymapper

for frontend
```bash
cd frontend
npm install
npm start
```

for backend
```bash
cd backend
npm install
npm start
```
